Spine Injection CPT Codes: 2026 Medicare Billing Guide

TFESI, interlaminar epidural, facet joint, and joint injection codes with NF vs facility payment spread analysis, utilization limits, bilateral modifier rules, and payer-specific denial patterns.

Non-QPP CF: $33.40Updated: March 2026Source: CMS CY 2026 PFS + LCD articles

Why injection economics matter

Spine injections are a critical revenue driver for orthopedic practices, particularly those with office-based fluoroscopy. The non-facility (NF) vs facility payment differential is substantial. A single-level lumbar TFESI (64483) pays $265 in the office vs $100 in the hospital outpatient setting, a spread of $165 per injection. This differential is the primary economic driver behind the growth of office-based injection suites. Understanding the NF/Fac spread for each injection type directly informs site-of-service decisions and practice profitability.

Medicare limits epidural injections to 4 sessions per spinal region per rolling 12-month period. Transforaminal epidurals are limited to 2 levels per session. Only one spinal region may be treated per session. These utilization parameters are enforced through Local Coverage Determinations and audited by MAC contractors.

Transforaminal epidural steroid injection (TFESI)

TFESI codes (64479-64484) describe injection of anesthetic and/or steroid via the transforaminal approach with fluoroscopic or CT guidance. Imaging guidance is bundled into the code; do not separately report fluoroscopy (77003). These are unilateral codes; for bilateral procedures, append modifier 50 (or use RT/LT line items per payer preference). The primary code (64483 for lumbar, 64479 for cervical) covers the first level; the add-on code covers each additional level. Do NOT report 64483 x 2 for two levels; use 64483 + 64484.

Interlaminar epidural steroid injection

Interlaminar ESI codes (62322-62323) describe injection via the interlaminar or caudal approach. 62322 is without imaging guidance, 62323 is with fluoroscopy or CT. Most MACs require imaging guidance (62323) for Medicare coverage; 62322 is frequently denied as not medically necessary. Interlaminar ESI is not a bilateral procedure; do not append modifier 50. Only one level per session for interlaminar approach.

Facet joint injection / medial branch block

Facet injection codes (64490-64495) describe injection of diagnostic or therapeutic agent into the facet joint or the nerves innervating it, with image guidance. These codes are reported per level, not per needle. Injecting multiple medial branch nerves at the same facet joint level = 1 unit. This is the single most common overcoding error identified by OIG audits: counting nerves instead of joint levels. Fluoroscopy/CT guidance is bundled; do not separately bill 77003.

Count facet JOINT LEVELS, not medial branch NERVES. Each lumbar facet joint is innervated by 2-3 medial branch nerves. Injecting all nerves at one joint level = one billable level. Reporting one code per nerve instead of per joint level is the pattern that triggered $4.2M in OIG-identified improper payments in a single MAC jurisdiction. The add-on code (64494) has a Medically Unlikely Edit (MUE) of 1 per date of service; you cannot report 64494 x 2 for a "third level." Use 64495 for the third and any additional levels.
Bilateral facet injections: Use modifier 50 on the primary code, not on add-on codes. For bilateral injections at L3-4: report 64493-50. For bilateral L3-4 + bilateral L4-5: report 64493-50 + 64494-50. Add-on codes follow the bilateral modifier of the primary code. ASC facilities must use RT/LT on separate lines instead of modifier 50. This varies by payer; always verify.

Joint and bursa injection/aspiration

CPT 20610 (large joint without ultrasound) and 20611 (with ultrasound) are commonly used for knee, hip, and shoulder injections in orthopedic practice. These are frequently performed on the same day as an E/M visit, requiring modifier 25 on the E/M code to indicate a significant, separately identifiable evaluation beyond the injection decision.

NCCI edit pairs and bundling rules

62323 + 64483 at the same spinal region on the same day: Rarely appropriate. Interlaminar and transforaminal epidural at the same lumbar region on the same date requires modifier 59/XS and strong documentation of distinct clinical indications for each approach. Most payers deny this combination. If both approaches are clinically needed, consider separate sessions.
Do NOT report 64483 x 2 for two levels. Use 64483 + 64484. This is one of the most common injection billing errors. 64483 is the primary code for the first level. 64484 is the add-on for each additional level. Reporting 64483 twice results in denial of the second unit. LCD limits: maximum 2 levels per session for TFESI.
Fluoroscopy (77003) is bundled into all injection codes. 64479-64484, 62322-62323, 64490-64495, and 20611 all include image guidance. Separately reporting 77003 for fluoroscopy with these codes is a duplicate billing error and an OIG audit flag.
KX modifier for DSNRB (diagnostic selective nerve root block). DSNRB is identically coded as TFESI (64479-64484). When performing a DSNRB (not therapeutic), append modifier KX to distinguish it from a therapeutic TFESI. Aberrant use of KX triggers focused medical review. If you performed a therapeutic TFESI, do NOT append KX even if the MAC denies for "missing modifier." This is a known LCD interpretation conflict with some MACs.

Common denial patterns

Medicare (all MACs)

5th epidural session in rolling 12 months denied (frequency limit)

Track rolling 12-month window PER SPINAL REGION. 4 sessions allowed per region. If the 5th injection is in a different spinal region, it may be billable. Maintain a tracking spreadsheet per patient per region.

Aetna

TFESI denied when prior injection did not achieve >50% relief for >3 months

Document pain relief percentage and duration after each injection using validated tools (VAS, NRS). If prior injection achieved less than 50% relief or lasted less than 3 months, Aetna requires reassessment before subsequent injection.

UnitedHealthcare Medicare Advantage

TFESI denied for "missing modifier" when KX is not appended

Known LCD interpretation conflict. UHC Medicare Advantage sometimes requires KX modifier on all TFESI claims, not just DSNRB. If therapeutic TFESI is denied for missing modifier, appeal citing that KX is only for DSNRB per CMS billing article. Escalate to MAC medical director if needed.

BCBS / Commercial

Facet injection at 3+ levels denied as exceeding medical necessity

Many commercial payers follow ODG guidelines limiting facet injections to 2 levels bilaterally. Document specific clinical indication for each level. Some plans require prior authorization for 3+ levels.

Frequently asked billing questions

Right L4 TFESI + left L5 TFESI: is this bilateral (modifier 50) or two levels (64483 + 64484)?

Two levels. Modifier 50 indicates the same procedure performed on both sides at the same level. Right L4 + left L5 = two different levels, so code 64483-RT + 64484-LT. If right AND left L4, that is bilateral at one level: 64483-50. If right and left L4 + right and left L5: 64483-50 + 64484-50.

Can I do cervical AND lumbar TFESI on the same day?

Medicare LCD states only one spinal region may be treated per session. Cervical and lumbar are different regions, so this would violate the one-region-per-session rule for Medicare. Some commercial payers allow multi-region same-day injections. Check the specific payer LCD before scheduling. If allowed, report 64479 + 64483-59 with documentation of distinct clinical indications for each region.

How many medial branch nerves = one facet level?

Each lumbar facet joint is innervated by 2-3 medial branch nerves. Each cervical/thoracic facet joint by 2 nerves. Injecting all nerves at one joint level = one billable level, regardless of needle count. Report 64493 for the first lumbar level (all nerves at that level combined), 64494 for the second level, 64495 for the third. Counting individual nerves as separate levels is the OIG-flagged overcoding pattern.

Can I bill an E/M visit (99214) on the same day as a TFESI?

Yes, if the E/M service is significant and separately identifiable beyond the decision to inject. Append modifier 25 to the E/M code (e.g., 99214-25 + 64483). The E/M documentation must stand on its own as a distinct evaluation (e.g., medication management, review of imaging, discussion of surgical options) separate from the injection procedure decision. If the only E/M work is deciding to do the injection, modifier 25 is not appropriate.
